What is Pulse (bpm)?

The calculated pulse is in beats per minute. This is implemented so the user can monitor his/her health more effectively. The data is stored every time, the cane is in use.
The meter above depicting data when above the threshold falls in red zone of the circular meter scale.

The average pulse rate of an elderly men is 70-73 bpm.
The average pulse rate of an elderly women is 73-76 bpm.
The average pulse rate while walking for an elderly men is 80-83 bpm and that for an elderly women is 83-86 bpm.
